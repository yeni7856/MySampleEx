2. 배너 광고 구현
- 게임 시작하면 배너광고 생성, 로드
- 메인메뉴에서 화면 상단에 배너광고 보여주기
- 옵션 누르면 옵션차에서는 배너광고 숨기기
- 다시 메인메뉴 나오면 베너광고 보여주기

3. 전면 광고 구현
- 게임 시작하면서 광고 요청하기
- 메인메뉴에서 크레딧를 누르면 전면광고 보여주기


4. 리워드 광고 구현 : 광고 시청 후 보상 지급
- 게임 시작하면서 광고 요청하기
- 메인메뉴에서 옵션을 누르면 보상형광고 보여주기
- 광고를 보면 보상으로 ammo 10개 지급 - 유저가 광고를 끝까지 보았을 경우 보상 지급

5. 광고보기 버튼 : 
- 광고보기 아이템
  - 가격 무료 0원
  - 샵 아이텝 등록 : 광고보기 아이템 
  - 샵 광고보기 아이템을 선택하면 버튼이 광고보기 버튼 활성화
  - 광고보기 버튼 누르면 광고 보여준다
  - 광고를 보고 나면 쿨 타임 적용(1~5분)
  - 쿨 타임 적용되면 광고보기 버튼 비 활성화
